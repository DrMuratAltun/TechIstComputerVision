# TechIstComputerVision

TechIstComputerVision, bilgisayarlı görme (computer vision) uygulamaları için geliştirilmiş bir projedir. Bu repo, görüntü işlemeden renk dönüşümüne, OpenCV kullanımından Gradio ara yüz uygulamalarına kadar çeşitli konularda örnek kod ve görseller barındırır. Amaç, görüntü işleme ile ilgili çalışmalara bir başlangıç noktası sunmak ve öğretici örnekler sağlamaktır.

## Özellikler
- **Görüntü Manipülasyonu:** Görüntüleri yeniden boyutlandırma, döndürme, kırpma gibi temel işlemler.
- **Renk Dönüşümleri:** Görsellerin farklı renk uzaylarında işlenmesi (ör. RGB'den GRAY'e dönüşüm).
- **OpenCV Tabanlı Fonksiyonlar:** OpenCV kütüphanesi kullanılarak çeşitli görüntü işleme uygulamaları.
- **Gradio Tabanlı Uygulamalar:** Web tabanlı uygulamalar geliştirmek için Gradio kullanımı.
- **Eğitici Uygulamalar:** Görselle ilgili temel ve ileri düzey konuları açıklayan Python dosyaları.

## İçerik
### Klasörler ve Dosyalar
- **src:** Projeye ait ana kaynak kodlarının bulunduğu klasör.
- **zoom_kodlar:** Çeşitli örnek uygulama kodları.
- **Görseller:** Proje kapsamında kullanılan JPEG/PNG görseller.
- **Uygulama dosyaları:** Her biri belirli bir görüntü işleme fonksiyonunu veya öğretici özelliği içeren Python dosyalar.

### Önemli Python Dosyaları
- **uygulama1_openCV_versiyon.py:** OpenCV versiyonunu kontrol eden bir başlangıç dosyası.
- **uygulama6_renk_uzayı_dönüşümü.py:** Görsellerdeki renk uzayı dönüşüm işlemlerini içerir.
- **uygulama12_gorsel_kirpma.py:** Görselleri belirli alanlara kırpmayı gösteren bir uygulama.
- **u5_gradio_interface_SB.py:** Gradio kullanarak basit bir web arayüzü örneği.
- **u6_RenkDegistirmeGradio.py:** Renk dönüşüm işlemleri için bir Gradio uygulaması.

## Gereksinimler
Bu projeyi çalıştırmak için aşağıdaki yazılım ve kütüphanelerin kurulu olması önerilir:
- Python 3.7 veya üzeri
- OpenCV
- Gradio

Ek olarak, `requirements.txt` dosyasında listelenen bağımlılıkları yüklemek için aşağıdaki komutu kullanabilirsiniz:
```bash
pip install -r requirements.txt
```
